    !DOCTYPE

    É uma diretiva HTML que especifica o tipo de documento (ou versão do HTML) que está sendo usado em uma página web.
    Ela é usada pelo navegador para interpretar corretamente o código HTML da página. A declaração !DOCTYPE deve ser a primeira coisa que aparece em um arquivo HTML e é seguida pelo nome do tipo de documento.

    html lang = "pt-Br"

    O lang usada para especificar o idioma do conteúdo de uma página da web. O atributo "lang" é usado para especificar o idioma do conteúdo da página. Por exemplo, "pt-BR" é usado para especificar o português do Brasil.
    Essa informação é importante para navegadores, motores de busca e outros softwares que precisam identificar o idioma da página para fornecer uma experiência mais apropriada ao usuário.
